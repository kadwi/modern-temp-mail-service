Domain Setup Instructions

DNS Records Configuration

1. MX Record
Name/Host: @ (or leave blank)
Type: MX
Priority: 1
TTL: 86400
Value: mail.yourdomain.com
2. A Record for Mail Server
Name/Host: mail
Type: A
TTL: 86400
Value: YOUR_SERVER_IP

Replace YOUR_SERVER_IP with your actual server IP address. Allow 24-48 hours for DNS changes to propagate fully.
